# Climate Risk and Global Stock Market Volatility

This project tests whether climate risk indicators help predict stock market volatility across global equity markets.

## Objective

The aim is to test whether climate-related risk measures contain useful information for forecasting future equity market volatility. The project focuses on whether these signals can improve portfolio risk monitoring and investment decision-making.

## Methodology

- Collected and cleaned monthly stock market volatility data across global equity markets
- Merged financial market data with climate risk indicators
- Applied PCA and PLS to reduce high-dimensional climate risk variables
- Built predictive regression models to test the forecasting power of climate risk signals
- Compared in-sample and out-of-sample forecasting performance
- Extended the analysis using machine learning models where appropriate

## Tools Used

Python, pandas, NumPy, scikit-learn, statsmodels, Excel

## Current Status

This is an ongoing MSc dissertation project. The repository currently includes the working code used for data cleaning, climate risk index construction, predictive regression modelling and forecast evaluation.
